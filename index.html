This is a basic Gulp setup.

The gulpfile.js will do the following things:

  • Process HTML files in ./src/
  • Process Jade files in ./src/templates/
  • Process Sass files in ./src/styles/
  • Process script files in ./src/scripts/
  • Process JS libraries in ./src/scripts/libs/
  • Create a server on http://localhost:8080
  • Watch that server for changes

It also lints the scripts, concatinates and minifies scripts into a single file and libraries into another. It processes and minifies Jade and HTML.

In order to run this, you will need NodeJS and Gulp. In your local project, install dependencies defined in package.json by running npm install.

You can find this setup at GitHub and me on @svdb.
